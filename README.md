# 🚀 Getting Started

Clone Repository

```bash
git clone https://github.com/yourusername/backend-series.git
```

Move into folder

```bash
cd backend-series
```

Install dependencies

```bash
npm install
```

Run Project

```bash
npm start
```

or

```bash
npm run dev
```
